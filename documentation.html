This page contains shortcode and functional information.

Admin Options Panel Overview ​

Currently, there are two options panels in the admin that offers extended functionality throught the theme.

Design Options ​

Background Textures

Like in the Customizer > 30 Lines > Theme Design Options panel, here you can also add/edit the Page and Header background images. There is also a 3rd Alt Background that is not currently used, however one can be added here and used throughout the site in Kadence Blocks that offer Dynamic Background.

TIP

Editing background images is probably better to do in the customizer, as you will be able to see that update in real time while editing. Saving it there will also update this section. Since Kadence Blocks syncs to global options and not customizer fields, it had to be set-up in both locations.

Desktop Navigation Patterns

Here you can choose to keep the navigation hidden in the overlay, or turn it off to have the links display in the top main menu. Overlay is the default.

Home Search Input

Multi-Property sites show the Property/Location search field on the home page by default. If you'd like to just show the button to link to the RentPress apartments page, activate this toggle and save.

Property Global Settings ​

Property Archive Slug

This will update the home page search button to the url that the search page lives. This also updates the links found in the breadcrumbs on the Single Property pages and their schema.

Property FAQ

These are global and will show up on all property pages.

Office Hours

This is for general areas of the site where the corporate office hours are needed. Single Property pages will still show the hours of that property.

Custom Property Sub-Nav Links

Property Sub Nav Links, this will add navigation links to the top of the single property pages. Can add classes like "is-cta", "is-cta-primary" ( this will alternate your chosen colors )

Search Input Parameters

These fields allow you to alter the main Search Input label and button text fields.

Customizer ​

This section outlines the customizer settings found at Customizer > 30 Lines

Customizer > 30 Lines > Global Options ​

This section contains fields used for various parts of the sites that are not pulling in specific property data.

  • Address - First Line
  • Address - Second Line
  • Phone Number

Customizer > 30 Lines > Theme Design Options ​

  • Header Texture Background

Changes the background texture that is the hero sections of pages and posts, as well as some banners

  • Page Texture Background

Changes the background texture throughout pages and applicable blocks

TIP

In a Kadence Row Block ( as well as others ), under Style, and Background Image, click Dynamic Background Image and Enable Dynamic Background Image, choose Background Image -> Site Custom Setting, and Custom Field -> Options: Page Background ( or Options: Header Background), so the Gutenberg Block background image stays in-sync with the chosen customizer images.

Font fields

  • Header Font
  • Normal Weight
  • Bold Weight
  • Normal Weight
  • Bold Weight

Customizer > 30 Lines > Footer Settings ​

This section controls some of the data found in the main footer.

  • Footer Background Image

This will place a background image to the whole footer area

  • Footer Logo

By default the footer logo will also be whats entered for the main navigation logo, however it can be overwritten here.

  • Footer Address

An override for the Global Options address

  • Footer Text Area

For miscellaneous text that needs to be added to the footer, this will populate under the address in the menu.

  • Additional Footer Icon

This will add another icon next to the Equal Housing and Accessibility icons in the footer.

  • Additional Footer Buttons

There are also 2 fields for adding buttons to the footer, these populate in the Connect With Us section.

Customizer > 30 Lines > Menu Overlay ​

Here, you can choose the Background Photo and centered Logo that appears in the Menu flyout overlay.

Shortcodes ​

Functional shortcodes designed to be used through the theme and pages.

Corporate Office Hours ​

[corp-office-hours]

Utilzes the office hours that are entered in the Property Global Settings area of the dashboard. This is used in general areas of the site, the Property Single pages pull the hours data in via RentPress and the hours in the feed.

Property Search ​

[property-search]

A search box that appears on out home page for multi-property sites. This will search through the sites properties by name and various address points. This can be used on other pages as well, however only one per page.

Team Members ​

[team_members]

Displays a grid of Team Members that are entered in the Team post type.

Customizer Shortcodes ​

These shortcodes will display the data entered in the cutomizer via shortcodes

Customizer > 30 Lines > Global Options ​

[address-line-1]
[address-line-2]
[phone]
[phone-link]

These all display the fields entered in here, and the phone-link option also wraps the number within a link.
